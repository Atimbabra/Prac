HTML(HyperText Markup Language)

Html is a standard markup language for documents designed to be displayed in a web browser and for creating web pages.

Html describes the structure of a web page. It consists of a series of elements that tell the browser how to display the content. It is often assisted by technologies such as Cascading style sheets and scripting languages such as Javascript.

roles of html in web development

html defines the structure of a webpage by using various elements like headings, paragraphs, lists and more which help organise the content logically making it easier for both users and search engines to understand the information presented.

html allows you to apply formatting to text, such as making text bold, italic, underline or using different font sizes and colors.

html enables creation of hyperlinks that connec diffferent webpages and resources. Hyperlinks allows users to navigate between pages and access external content.

html llows you to embed images , videos, audios and pther media types into web pages, enhancing the visual and interactive experience for users.

html provides semantic elements like headings, paragraphs and sections that convey the meaning and structure of the content to both humans and search engines.

The timeline of html

Here's a timeline of key events in the development of HTML:

  • 1989: Tim Berners-Lee, a British computer scientist, proposed the concept of the World Wide Web and outlined the basic idea of HTML.
  • 1990: Berners-Lee created the first web browser/editor called the "World Wide web" and introduced the first version of HTML, which was very basic.
  • 1993: HTML was extended to include features like tables and forms in HTML 2.0.
  • 1995: HTML 3.0 was published as a working draft, introducing more structural elements and improvements.
  • 1997: HTML 4.0 was published as a recommendation by the W3C, adding more features like frames, style sheets, and scripting support.
